## What Is the Vasconcelos Method?

The **Método Vasconcelos** is the first comprehensive framework that integrates **six schools of theatrical improvisation** (Johnstone, Del Close, Spolin, Napier, UCB, Second City), **Freirean dialogical pedagogy**, **Feynman's simplification technique**, and **evidence-based communication science** into a unified clinical method for transforming medical practice.

**Cost of implementation: R$ 0 per consultation.**
**Training required: 20 hours over 8 weeks.**
**Evidence level: Level I (meta-analyses, Cochrane reviews, JAMA, Lancet, BMJ).**

## The Core Innovation: REGISTRO VASCONCELOS

A 7-field documentation system designed to **replace SOAP** (1968):

| # | Field | What It Captures |
|---|-------|-----------------|
| N | **NARRATIVA** | Patient's own words — verbatim |
| C | **CONTEXTO** | Social determinants, life events, support |
| E | **EXAME** | Physical exam + RASTREIO-7 screening |
| M | **MODELO** | What the patient believes is wrong |
| S | **SÍNTESE** | Integrated clinical reasoning |
| A | **ACORDO** | Shared plan + what was NOT done (quaternary prevention) |
| T | **TRANSMISSÃO** | Structured handoff with complexity tier |

**SOAP = 4 fields. Registro Vasconcelos = 7 fields. 2 extra minutes = 500% more information.**
